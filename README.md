Quick demo of implementing Brewit's embed chat

## Getting Started

### Install

```bash
npm install
```

### Run

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
